Requirements
------------
 - PHP >= 8.0
 - Laravel >= 8.0

Installation
------------

``` bash
composer require peidgnad/peidmin
```

Configurations
------------

The file `config/peidmin.php` contains an array of configurations, you can find the default configurations in there.

Testing
------------

``` bash
composer test
```
